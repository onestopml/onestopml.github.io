# Smart Query
With smart query, we have a lot of advanced ways to filter and select samples in our dataset.
## Query Bar
To query samples with some specific conditions, go to "Query bar" (as shown in [Figure 1a](#fig1)) 
and type some query conditions in Renex query format. 
The "Query bar" are integrated with autocomplete feature which suggests relevant query strings 
as shown in [Figure 1b](#fig1)
