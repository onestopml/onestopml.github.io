Working with Models#

Each user can manage their models and permissions (keeping private or sharing to everyone). A model could be shipped with its own description, tags and multiple versions. Each version also contains its own information such as description, tags and the **schema**. To start, we do: 
1import renex
2import logging
3import pprint 
4pp = pprint.PrettyPrinter(indent=4)
5
6logging.getLogger('renex').setLevel(logging.INFO)
7
8renex.init(username='####', password='####', host='http://localhost:9000')

We will support later

List models#

The object renex.models contains all of your models. It is a dictionary-like object, thus we can get walk-through as following:

1print('Number of models:', len(renex.models))
2for name, model in renex.models.items():
3    print('=' * 40)
4    print('model name:', name)
5    print(model.json())
